**What you are seeing**
A faucet request modal for test tokens on Sei Testnet. It explains request limits and provides a single action to submit a request.

**Key regions**
- Warning banner describing the 24-hour request cadence.
- Primary `Request` action and a cancel action.

**Notable behaviors**
- The modal is only relevant in testnet contexts.
- The request is rate limited to prevent abuse.
