**What you are seeing**
The swap interface (`/en/swap`) for exchanging supported assets with a quote-based flow.

**Key regions**
- Input/output asset fields with amount entry.
- Quote and route details for the current swap path.
- Execution controls to submit or review the swap.

**Notable behaviors**
- Quotes refresh as token pair or amount changes.
- Submission requires a connected wallet and sufficient balance.
