**What you are seeing**
The `Trading Preferences` modal for RPC endpoint configuration used by the trading session.

**Key regions**
- Current mode and network status section.
- RPC input field for a custom endpoint.
- Community RPC example shortcuts.
- Action row with cancel, reset, and save controls.

**Notable behaviors**
- Endpoint settings are applied to wallet connectivity behavior.
- Save remains disabled until input is valid and different from current settings.
