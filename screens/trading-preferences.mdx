**What you are seeing**
A modal for configuring RPC connectivity used by the trading interface. Users can set a custom RPC endpoint and pick from community examples.

**Key regions**
- Status chip showing current RPC mode and network (e.g., `Sei Testnet`).
- Input field for a custom HTTPS RPC endpoint.
- Example RPC buttons for quick selection.
- Action buttons for cancel, reset, and save.

**Notable behaviors**
- Changes are stored locally and applied on wallet connection.
- The modal enforces HTTPS for security.
