![Language menu](/images/app.m1.markets_en%20%284%29.png)

**What you are seeing**
The trade screen with the language selector open, showing available locales for the UI.

**Key regions**
- Locale menu in the top-right with flags and language labels.
- Background trade view remains visible for context while changing languages.

**Notable behaviors**
- Selecting a language switches the UI text without leaving the current screen.
