This support guide helps you resolve common issues on MACH1 before opening a ticket.

Response times for tickets are usually within 48 hours and may be longer on weekends.

## Recommendations

1. Search this support page and the docs for your issue first.
2. Check wallet connection, network, and transaction status before retrying actions.
3. If the issue persists, contact support at `support@m1.markets` with:
   - your wallet address,
   - approximate timestamp,
   - transaction hash (if available),
   - short description of expected vs actual result.
4. Never share private keys, seed phrases, recovery phrases, or one-time codes.

## Common issue types

- Connectivity and wallet connection issues
- Deposits or transfers not appearing
- Withdrawal delays or failures
- Trade outcomes that look incorrect
- Third-party app or integration issues

## Important reminders

1. Use only official MACH1 domains, including `m1.markets`.
2. Ignore unsolicited direct messages claiming to be support.
3. Verify every URL before signing any wallet message or transaction.
4. Keep wallet software updated and protect access with strong security settings.

## Security notice

MACH1 support will never ask for your private key, seed phrase, or full wallet backup.
